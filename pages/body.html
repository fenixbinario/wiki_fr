MVVM & MVC

En el body vamos a declarar las capas de nuestra aplicacion mediante el atributo layer="", para este ejemplo queremos una web con la página de incio, otra que la llamaremos Rick y otra Morty.

<body layer="app rick morty">

Para este ejemplo vamos a crear tres vistas/páginas <app> <rick> <morty> para mostrar el poder de fr.js puedes ver el resultado en el DEMO 03

load=""

Unas de las necesidades de nuestros desarrollos es que sean escalables y mantenibles, para ello unas de las necesidades es dividir por partes nuestro código. con el atributo load="" nos permite añadir un trozo de código html dentro de nuestro código existente. Para este ejemplo vamos a incrustar tres archivos app.html, rick.html y morty.html. Mira el resulta aquí DEMO 04

view=""

Este atributo view="" nos permite mostrar la capa de nuestra aplicación que queremos hacer visible. Mira el resulta aquí DEMO 04.

lang=""

En nuestra carpeta raiz colocamos los archivos .json con los idiomas nombrandolos con el el código de idioma y pais es-ES.json en-EN.json en-UK.json.

Este atributo lang="" se puede colocar en cualquier elemento del DOM para ello solo hay que poner el nombre de la clave que aparece en el .json . Mira el resulta aquí DEMO 05, estos son los .jon es.
