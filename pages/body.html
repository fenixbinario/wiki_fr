MVVM & MVC

En el body vamos a declarar las capas de nuestra aplicacion mediante el atributo layer="", para este ejemplo queremos una web con la página de incio, otra que la llamaremos Rick y otra Morty.

<body layer="app rick morty">

Para este ejemplo vamos a crear tres vistas/páginas <app> <rick> <morty> para mostrar el poder de fr.js puedes ver el resultado en el DEMO 03

host="" headers="" role=""

Una vez tenemos declarado nuestro elemento <rick-morty></rick-morty> vamos a configurar el servidor añadiendo los atributos:

  • host="" para decirle el nombre del servidor y cuales seran las cabeceras de las peticiones.
  • headers="" se define las cabeceras de las peticiones y despues se declaran, por defecto se puede dejar vacio.
  • role="" se define el role de acceso y privilegios al servidor, por defecto se puede dejar vacio.

En este ejemplo queremos obtener datos de la API de Rick&Morty en formato JSON por eso hemos declarado en headers="" que nuestras peticiones tendrán una cabecera de Content-Type para más adelante especificarle que será application-json.
