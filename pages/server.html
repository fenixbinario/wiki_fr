El elemento server <server> es una palabra reservada utilizada dentro del body.
Dentro del elemento <server></server> podemos declarar los servidores a los que vamos a conectarnos.

Por ejemplo si queremos conectarnos a la API de https://rickandmortyapi.com/api podemos declararlo como un elemento dentro del <server></server> lo llamaremos <rick-morty></rick-morty> para este ejemplo.
