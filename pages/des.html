FENIX RELOAD

WEB 3.0

Fénix ReLoad es un motor de javascript orientado a la WEB 3.0, renderiza el Front-End en el lado del cliente, mejorando las capas de integración entre nodos , servidores, externalización de procesos y bases de datos entre los integrantes que consumen el servicio o lo utilizan como intermediarios. Permite desarrollar aplicaciones escalables en la WEB 3.0 de manera fácil, rápida, e hiperconectada. Permite separar el modelo de negocio del desarrollo del código. Permite separar el modelo vista-controlador ordenadamente sin mezclar tareas entre front-end y back-end. Permite su portabilidad a diferentes plataformas de escritorio, móvil, web y aquellas que soporten tecnologías de navegador web.


Seguridad

Token , Auth, User y Pass

Conexiones a servidores mediante los estandares de seguridad, permitiendo caducidad de sesion, roles, autorizaciones etc...


Conexiones Multiples

Unas de las caracteristicas es poder establecer conecxiones a la vez con muchas API, websocket y MQTT al mismo tiempo, sin tener que configurar ni escribir código especifico para cada unas de nuestras conexiones.

API

Conexión a una o varias API's mediante una sola línea de código se puede conectar a un servidor desde el propio marcado de HTML, siendo capaz de realizar GETTERS y SETTERS sin escribir ni una sola línea de javascript.
Desde el propio marcado de HTML podemos hacer hacer todo los METHODGET POST PUT PATCH DELETE y manejar las respuestas pintando nuestro DOM sin tener que escribit ni una sola línea de código.

WEBSOCKET

Conexión a uno o varios servers WS con una sola línea en HTML, además recibir y enviar datos de los eventos.

MQTT

Conexión a uno o varios servers MQTT con una sola linea en HTML, además de poder suscribirte y publicar desde el DOM sin tener que desarrollar un controlador en javascript.


Flexible

MVC y MVVM

Fénix Reload permite facilmente desplegar sus desarrollos tanto con las metodologías MVC y MVVM.
El objetivo es desarrollar en WEB3 de manera fácil, mantenible y escalable.
